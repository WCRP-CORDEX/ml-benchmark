# Submission

Submission of results for online benchmarking requires the registration of the particular method used. ***

Once registration is completed details will be forwarded to upload a single zipped file including contributions with the methods for the different experiments. 

An illustrative scripts helping to generate the file is provided.

- **`submission.py`**: It runs predictions for all available test predictor files and organizes the outputs according to the required submission structure. It uses official templates from `format_predictions/templates/` to ensure correct spatial coordinates and variable attributes.

For additional details on submission formatting, refer to the [official documentation](https://docs.google.com/document/d/1zM6Aza1glzBJ8Ewvhu0_0vcm5dvctJ65Bypt1gnN2ck/edit?usp=sharing).
